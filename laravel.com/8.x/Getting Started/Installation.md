# Meet Laravel
- expressive, elegant syntax
- thorough dependency injection: 你在框架里跳舞，像android和iOS一样；框架调你的代码，而不是你调框架的代码
- expressive database abstraction layer
- queues and scheduled jobs
- unit and integration testing
- real-time events
